Frågor och svar
Hur kan du som student söka examensarbete?
Som student kommer du ha möjlighet att söka efter valfria kriterier, samt se nya tillagda examensarbeten.

Kan student se allt information om ett examensarbete?
Ja, som student kan du få fram all information som krävs vid sökning av exjobb när du trycker på ett sökalternativ, du kommer även som student kunna ansöka valt exjobbet.

Kan du som student ansöka om flera examensarbeten?
Ja, som student kan du ansöka flera examensarbeten samtidigt och dina sökta exjobb kommer visas under rubriken "Startsida".

Hur kan man som student ändra i profilen?
Som student kan du ändra all information under rubriken "Profil".

Hur kan man som student ändra lösenord?
Du kan ändra lösenord endast om du är inloggad som student och trycka på knappen "ändra" med minst 6 karaktär och måste matcha.

Om TCM

Företaget TCM består av Petter Nordin, Edvin Esselin, Fredrik Lidström, Choekyi Choekyi,Pascal Touma och Edin Jelecevic.
Företaget fick i uppdrag av Örebro Universitet att utveckla en applikation för Dataingenjörerprogrammet där studenter
skulle kunna söka och ansöka exjobb.
